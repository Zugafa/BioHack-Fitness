# BioHack Fitness | Optimize Your Biology

**BioHack Fitness** este un proiect de tip „commercial website" dezvoltat pentru materia **Tehnici WEB**, în cadrul **Facultății de Matematică și Informatică, Universitatea din București**. Conceptul se bazează pe un centru de recompoziție corporală cu estetică cyberpunk și underground.

## 🧬 Concept & Obiective
* **Design Futuristic**: Interfață de tip "dark mode" cu accente neon (albastru electric/magenta).
* **Bio-Hacking**: Promovarea unui stil de viață bazat pe tehnologie și optimizare biometrică.
* **Structura Site-ului**: 5 secțiuni principale — The Mainframe, Protocols, The Lab, Access Point și User Node (Jack In).

## 🗂️ Structura Site-ului
- **Mainframe**: Pagina de impact vizual, manifestul BioHack și informații generale.
- **Protocols**: Sistemul de abonamente, de la Baseline (Human) până la Specimen Zero.
- **The Lab**: Echipa de specialiști (Cyber-Coaches, Bio-Analysts, Heart-Boosters) și zonele de antrenament.
- **Access Point**: Contact, localizare și rețele sociale.
- **User Node (Jack In)**: Zona securizată pentru membrii comunității SYNTH.
